# 🎆 Учёт пиротехники на складе

Простое и удобное веб-приложение для управления запасами пиротехники. Работает прямо в браузере на телефоне или компьютере.

---

## ✨ Возможности

- 🏭 **Несколько складов** — создавайте и переключайтесь между разными точками хранения
- 📦 **Учёт товаров** — добавляйте наименования, категории и количество
- 📏 **Гибкие единицы** — поддерживаются шт, кг, коробки, упаковки, метры
- ⏰ **Контроль сроков** — предупреждения о товарах с истекающим сроком годности
- 💾 **Сохранение данных** — всё хранится в памяти браузера (LocalStorage)
- 📊 **Экспорт/Импорт** — возможность сохранить резервную копию данных
- 📱 **Mobile First** — интерфейс оптимизирован для смартфонов

---

## 📋 Как начать работу

1. При первом запуске создайте свой первый склад
2. Добавьте товары (название, категория, количество, срок годности)
3. Используйте поиск и фильтры для быстрого нахождения товаров
4. Обновляйте количество кнопками `+` и `-`

---

## 🛠 Технологии

- HTML5
- Tailwind CSS (через CDN)
- Vanilla JavaScript (ES6+)
- LocalStorage для хранения данных

---

## ⚠️ Важно

- Данные хранятся **локально в браузере**. Если очистить кэш — данные пропадут.
- Рекомендуется периодически делать **Экспорт данных** в настройках.
- Соблюдайте правила хранения пиротехники!
